pdfgithub.com

bogdannbv/pdfgithub.com

View PDF files from GitHub directly in your browser. Take any GitHub URL that points to a PDF file and put pdf in front of github.com.

1. Find a PDF Open a PDF file in a GitHub repository.
2. Edit the URL Change github.com to pdfgithub.com.
3. View it The file opens in the browser instead of downloading.
Example URL
            
                @@ prefix the domain with pdf @@
                - https://github.com/bogdannbv/pdfgithub.com/blob/main/example/file.pdf
                + https://pdfgithub.com/bogdannbv/pdfgithub.com/blob/main/example/file.pdf
Tip: the URL should end in .pdf.
